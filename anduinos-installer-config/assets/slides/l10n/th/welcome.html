ยินดีต้อนรับสู่ AnduinOS

ขอบคุณสำหรับการติดตั้ง AnduinOS!


AnduinOS คือการแจกจ่ายลินุกซ์แบบกำหนดเองที่พัฒนาบนพื้นฐานของ Ubuntu ออกแบบมาให้ใช้งานง่ายและคุ้นเคย ไม่ว่าคุณกำลังเปลี่ยนจาก Windows หรือ macOS มาใช้งาน ด้วยการมี แหล่งเก็บแพ็กเกจเฉพาะของตัวเอง เหนือระบบนิเวศซอฟต์แวร์ที่กว้างใหญ่ของ Ubuntu คุณจะได้รับข้อดีทั้งสองอย่าง: ความเข้ากันได้ที่มั่นคงและรายละเอียดที่ผ่านการปรับแต่งเพิ่มเติมสำหรับเดสก์ท็อป เรายินดีที่จะพาคุณไปเยี่ยมชม!
