Três camadas de software, uma experiência fluida

O AnduinOS utiliza uma abordagem limpa de três camadas para a gestão de pacotes. O sistema base confia nos pacotes Ubuntu para um núcleo estável, enquanto os componentes específicos do AnduinOS são geridos através do Apkg — a nossa própria ferramenta de pacotes leve.

Para aplicações destinadas aos utilizadores, substituímos o Snap pelo Flatpak para entrega de aplicações isoladas (sandboxed) e compatíveis com várias distribuições. Isto mantém as suas aplicações isoladas do sistema, poupa espaço em disco e dá-lhe acesso a um catálogo massivo. Naturalmente, o AppImage e o tradicional APT estão sempre disponíveis quando precisar deles.
