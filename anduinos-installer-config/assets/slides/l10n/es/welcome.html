Bienvenido a AnduinOS

¡Gracias por instalar AnduinOS!


AnduinOS es una distribución de Linux basada en Ubuntu personalizada, diseñada para ser familiar y fácil de usar, ya sea que procedas de Windows o macOS. Con su propio repositorio de paquetes además del vasto ecosistema de software de Ubuntu, obtienes lo mejor de ambos mundos: una compatibilidad sólida y un acabado adicional para el escritorio. ¡No podemos esperar para mostrarte todo!
